Amazon EBS uses **AES-256** encryption and integrates with **AWS Key Management Service (KMS)**:

- A **data key** is generated by AWS KMS.
    
- This data key is then encrypted using a **KMS key** (either AWS-managed or customer-managed).
    
- The encrypted data key is stored with the volume metadata.
    

You can use:

- **AWS-managed keys** (default, alias: `aws/ebs`)
    
- **Customer-managed keys** for more control (e.g., rotation, disabling)
